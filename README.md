# Dapper User Order Management API

A RESTful ASP.NET Core Web API built using Dapper and SQL Server for managing Users and Orders.

## Features

- ASP.NET Core Web API
- Dapper Data Access
- SQL Server Database
- Stored Procedures
- Dependency Injection
- CRUD Operations
- Swagger Integration

## Technologies Used

- ASP.NET Core
- C#
- Dapper
- SQL Server
- Swagger/OpenAPI
